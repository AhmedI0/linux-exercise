# Linux Commands and Shell Scripting Learning Repository

This repository documents structured learning of Linux commands and shell scripting through hands-on practice and real-world exercises.

## Structure

- 00_shell_for_hpc → Previous HPC-related shell work
- 01_linux_basics → Basic Linux concepts and commands
- 02_linux_commands → Core command-line tools
- 03_shell_scripting → Bash scripting fundamentals
- 04_advanced_bash → Advanced scripting and automation
- labs → Hands-on labs from the course
- notes → Consolidated notes

## Goal

To build a strong foundation in Linux and shell scripting for:
- High-performance computing (HPC)
- Data Analysis
- Bioinformatics pipelines
- Automation and data analysis
