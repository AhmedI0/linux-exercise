# Linux Distributions

## Learning Objectives

- Describe what a Linux distribution is
- Differentiate between common Linux distributions
- Identify use cases of popular Linux distributions

## What Is a Linux Distribution?

A Linux distribution, or distro, is a specific version or flavor of the Linux operating system. Every Linux distribution uses the Linux kernel, but each distro may include different tools, commands, graphical interfaces, package managers, and support models.

## What Makes Distros Different?

Linux distributions can differ by:

- Default utilities and system tools
- Graphical user interface
- Shell and command support
- Package management system
- Community or commercial support
- Long-term support or rolling release model

## Common Linux Distributions

### Debian

Debian is one of the oldest Linux distributions. It is known for stability, reliability, open-source principles, and strong server usage.

### Ubuntu

Ubuntu is based on Debian and is developed by Canonical. It is widely used for desktops, servers, cloud systems, and IoT.

### Red Hat Enterprise Linux

RHEL is enterprise-focused and managed by Red Hat, an IBM subsidiary. It is commonly used in business and production environments.

### Fedora

Fedora is sponsored by Red Hat and often includes newer technologies. It is community-driven, secure, and actively developed.

### SUSE Linux Enterprise

SUSE Linux Enterprise is used in server and desktop environments. It supports multiple architectures, including ARM.

### Arch Linux

Arch Linux is highly customizable and follows a do-it-yourself approach. It is useful for advanced users who want full control of their system.
