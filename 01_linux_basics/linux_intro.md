# Introduction to Linux and Unix

## Learning Objectives

- Explain what an operating system is
- Describe the origins of Unix and Linux
- List important features of Linux and Unix

## Key Notes

An operating system is software that manages computer hardware and resources and allows users and applications to interact with the hardware.

Unix is a family of operating systems that began in the 1960s at AT&T Bell Labs. It was later rewritten in C in the 1970s, which made it portable across different hardware systems.

Linux is a Unix-like family of operating systems. It was developed as a free and open-source alternative to Unix.

## Important Timeline

- 1960s: Unix created at AT&T Bell Labs
- 1970s: Unix rewritten in C
- Late 1970s: BSD developed at UC Berkeley
- 1980s: GNU project started
- 1991: Linus Torvalds developed the Linux kernel
- 1992: GNU tools and Linux kernel began forming complete Linux operating systems
- 1996: Tux penguin became the Linux mascot

## Key Linux Features

- Free and open-source
- Multi-user
- Multitasking
- Portable across many hardware platforms
- Widely used in servers, supercomputers, cloud systems, Android devices, and personal computers

## Important Terms

- Kernel: Core part of the operating system that communicates with hardware
- GNU: Free and open-source Unix-like tools
- Linux: Kernel originally developed by Linus Torvalds
- Distribution: A complete Linux operating system built around the Linux kernel
