# Linux Terminal and Directory Navigation

## Purpose

This section covers basic Linux terminal navigation commands used to move around the filesystem and inspect directories.

## Essential Commands

```bash

pwd  #shows the present working directory

ls #lists the files and folders in current driectory

ls -l #shows files and folders in long format

ls -a #shows hidden files too

cd folder_name #moves into the directory

cd.. #moves one directory up

cd/ #moves to the root directory

cd~ #moves to home directory

clear #clears the terminal screen


## slash usage

/ #Represents the root directory or separates folders in a path.

## Tab completion

cd Doc<Tab>   #If only one match exists, the terminal completes
              # it automatically. If multiple matches exist, pressing
              # Tab twice shows possible options.

history # command to show previously ran commands
